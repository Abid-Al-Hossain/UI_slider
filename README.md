# UI Foundry Range Slider Studio

Standalone, self-contained Range Slider component generator built for marketplace sale and future UI Foundry SaaS merge-readiness.

## Features

- Field metadata: id, name, title, tabIndex, dir, lang, autocomplete where native
- Validation: required, disabled, readonly-safe states, helper, error, success, description
- Input affordances: placeholder, prefix/suffix, clear action, keyboard hints, min/max/step where native
- State preview: default, focus, disabled, invalid, filled, empty
- React-only export: preview, visible code, copy, and download are sourced from one payload.
- Local shared controls: no runtime imports from other component projects.

## Commands

```bash
npm install
npm run typecheck
npm run lint -- --quiet
npm run build
npm run dev
```
